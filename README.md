## Ajouter une nouvelle page

1. Crée un dossier au nom du slug (sans accents, sans espaces, en minuscules) : `nouveau-slug/`
2. Mets-y au minimum un `index.html` (et tes assets)
3. Commit + push → la page est en ligne immédiatement sous `go.mediboxantillesguyane.com/nouveau-slug/`
